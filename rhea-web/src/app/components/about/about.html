Welcome to RheaWeb

Welcome to the RHEA. Some user caveats are the following:

  • To visualize the constraints as a tree you have to select its corresponding text.
  • The order in wich you write constraints must be done as a tree model, so first, the root, and then if it has two childrens the first one should be the one who came before the logical preposition. Example: (A ^ B) will be written as ^AB.

Tutorial How to use the tool

  • Load the FM you want to edit
  • Select Features using the button
  • To save the progress remember to download the file at the end.
  • Before downloading check in the interoperability matrix if the language can support the model.
  • If any of the rows is orange, then use the refactoring provided to allow that value
